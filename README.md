# Okuyuki-UI-Playground

3Dの奥行き移動にマウスホイールを使ったUIのPlayground

## Demo

![Okuyuki-UI-Playground demo](./doc/demo.gif)

## 使い方

### 移動

- ドラッグで画面に平行移動
- ホイールで奥行き移動

- shift keyで奥行き移動が1/10

- 他のオブジェクトの軸にスナップ
- m keyで切り替え

### 回転

- ドラッグでトラックボール式回転
- ホイールでローカルy軸回転

- ctrl keyでスナップ回転
