# Okuyuki-UI-Playground

3Dの奥行き移動にマウスホイールを使ったUIのPlayground

## Demo

![Okuyuki-UI-Playground demo](./doc/demo.gif)

## 使い方

### 移動

- ドラッグで画面に平行移動
- ホイールで奥行き移動

- shift keyで奥行き移動が1/10
- ctrlで他のオブジェクトの軸にスナップ
- ctrl + shiftで一定間隔でスナップ
![alt text](<doc/2026-04-05 23-47-22.gif>)
![alt text](<doc/2026-04-06 01-19-55.gif>)

- 他のオブジェクトの軸にスナップ
- m keyで切り替え

### 回転

![rotate](<doc/2026-04-06 01-27-59.gif>)

- ドラッグでトラックボール式回転
- ホイールでローカルy軸回転

- ctrl keyでスナップ回転
